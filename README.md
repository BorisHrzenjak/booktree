# BookTree

A Chrome extension that opens your bookmarks as a compact, searchable tree graph.

## Install locally

1. Open Chrome and go to `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this project folder.
5. Click the BookTree extension button to open the bookmark graph in a new tab.

## Features

- Opens from the extension toolbar button.
- Reads Chrome bookmarks with the `bookmarks` API.
- Collapsible folder tree.
- Compact SVG graph with pan, zoom, fit, and reset controls.
- Search by bookmark title and URL.
- Search filters to matching branches.
- Left-click opens a bookmark in the same tab.
- Ctrl-click or Cmd-click opens a bookmark in a new tab.
